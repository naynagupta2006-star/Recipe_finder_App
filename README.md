# 🍲 Recipe Finder Android App

Recipe Finder is an Android application that helps users quickly discover different recipes and explore new dishes. The app provides a simple and user-friendly interface to search and view recipes.

## 🚀 Features
- Search for different recipes
- Clean and simple user interface
- Fast loading recipe results
- Easy navigation

## 🛠️ Technologies Used
- Kotlin
- Android Studio
- Jetpack Compose
- API for recipes

## 📱 How the App Works
1. User opens the Recipe Finder App.
2. The app fetches recipe data.
3. Recipes are displayed in a list with images.
4. Users can explore different dishes easily.
